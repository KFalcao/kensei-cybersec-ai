# Semana 02 - Introdução à Programação em Python

Esta pasta contém os scripts e materiais desenvolvidos durante a segunda semana do curso de cibersegurança com IA. Os exercícios focam em conceitos básicos de programação em Python, aplicados a cenários práticos.

## Scripts Desenvolvidos

### 1. `celsius_to_fahrenheit.py`
Um conversor de temperaturas que permite converter valores entre Celsius e Fahrenheit. O usuário escolhe o tipo de conversão e insere a temperatura desejada.

### 2. `gerador_senhas.py`
Gerador de senhas seguras. Permite gerar uma senha única ou um lote de 5 senhas, com opções para incluir letras maiúsculas, números e símbolos. As senhas em lote são salvas em um arquivo com data.

### 3. `hello.py`
Script simples que solicita o nome do usuário (mínimo 3 letras) e exibe uma saudação em maiúsculo.

### 4. `lista_de_compras.py`
Gerenciador de lista de compras. Permite adicionar, visualizar e remover itens da lista. A lista é salva e carregada automaticamente de um arquivo `lista_compras.txt`.

### 5. `organizador_arquivos.py`
Organizador de arquivos que classifica arquivos no diretório atual em subpastas baseadas em suas extensões:
- `imagens/`: Arquivos de imagem (.jpg, .png, etc.)
- `docs/`: Documentos (.txt, .pdf, .doc, etc.)
- `videos/`: Vídeos (.mp4, .avi, etc.)

### 6. `quiz_cybersecurity.py`
Quiz interativo sobre cibersegurança com 5 perguntas. Inclui perguntas sobre phishing, firewall, autenticação de dois fatores, senhas fracas e ransomware. Usa timeout para as respostas.

## Estrutura de Pastas

- `docs/`: Pasta para armazenar documentos organizados (atualmente vazia).
- `imagens/`: Pasta para armazenar imagens organizadas (atualmente vazia).
- `videos/`: Pasta para armazenar vídeos organizados (atualmente vazia).

## Como Executar

Para executar qualquer script, navegue até a pasta `semana-02` e use o comando:

```bash
python nome_do_script.py
```

Certifique-se de ter Python instalado no sistema.
